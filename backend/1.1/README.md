### Problem Solving 1.1

ใช้ภาษาอะไรก็ได้เพื่อแก้ไขปัญหา

มีหินอยู่จำนวน N ก้อน แต่ละก้อนสามารถเป็นสีแดง R เขียว G หรือน้ำเงิน B หากนำหินทั้งหมดไปสุ่มวางเรียงกันเป็นแถวเดียว จงหาจำนวนของคู่หินที่มีสีเดียวกันและอยู่ติดกัน

Input
แบ่งออกเป็น 2 บรรทัดโดยบรรทัดแรกคือค่า N แสดงถึงจำนวนหินที่มีและบรรทัดที่สองจะเป็นลำดับสีของหินที่ถูกเรียงไว้แล้ว
Output
ตัวเลขแสดงจำนวนคู่หินที่มีสี่เดียวกันและอยู่ติดกัน

ตัวอย่าง

```sh
Input
3
RRG
Output
1
```

```sh
Input
5
RRRRR
Output
4
```

```sh
Input
4
BRBG
Output
0
```
