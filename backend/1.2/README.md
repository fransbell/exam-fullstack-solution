### Problem Solving 1.2

ใช้ภาษาอะไรก็ได้เพื่อแก้ไขปัญหา

ประเทศ XYZ มีธนบัตรทั้งหมดอยู่ 5 ชนิดคือ 1, 5, 10, 20 และ 100 ลุงพรชัยมีเงินอยู่ในธนาคาร N บาท หากถอนเงินออกมาทั้งหมดจะได้ธนบัตรจำนวนน้อยที่สุดจำนวนเท่าไร

Input
บรรทัดแรกคือค่า N ซึ่งแสดงถึงจำนวนเงินของลุงพรชัย
Output
แสดงจำนวนธนบัตรที่น้อยที่สุดที่ลุงพรชัยจะได้รับ

ตัวอย่าง

```sh
Input
125
Output
3
```

```sh
Input
43
Output
5
```

```sh
Input
1000000000
Output
10000000
```
